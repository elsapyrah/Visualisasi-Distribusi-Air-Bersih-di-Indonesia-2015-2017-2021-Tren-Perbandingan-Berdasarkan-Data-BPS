# Deskripsi Proyek
Proyek ini merupakan salah satu tugas mata kuliah Visualisasi Data dan Informasi dengan mengembangkan dashboard interaktif berbasis data BPS (2015, 2017â€“2021) untuk memvisualisasikan pola distribusi akses air bersih di Indonesia. Visualisasi mencakup tren perubahan, kualitas air, serta perbandingan antarwilayah, sehingga dapat membantu pemangku kebijakan memahami dinamika permasalahan dan merumuskan strategi peningkatan pelayanan air bersih, khususnya di pedesaan. Proyek ini menggunakan tools Google Colab dan Google Looker Studio.
